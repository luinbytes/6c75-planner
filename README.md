# 6c75-planner

A minimalist CLI-style task manager built with React and Framer Motion, featuring a clean terminal-inspired interface.

## Features

- Terminal-inspired interface
- Command-line task management
- Smooth animations and transitions
- Modern, minimal aesthetic
- Responsive design

## Tech Stack

- React
- TypeScript
- Framer Motion
- TailwindCSS

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/6c75-planner.git
cd 6c75-planner
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Available Commands

Type `help` in the terminal interface to see available commands.

## Development

This project uses:
- Vite for build tooling
- TypeScript for type safety
- TailwindCSS for styling
- Framer Motion for animations
